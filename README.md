# Camino Compañero

Website voor Camino Compañero — begeleide voorbereiding op de Camino de Santiago.

## Structuur
- `index.html` — de volledige website (één pagina)
- `images/` — alle foto's (eigen werk Edwin)

## Publiceren via GitHub Pages
1. Ga naar Settings → Pages in deze repository
2. Kies bij "Source": Deploy from a branch → `main` → `/ (root)`
3. De site staat binnen een minuut op `https://<gebruikersnaam>.github.io/<repo-naam>/`
